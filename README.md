Steps -

1. Download Zip File
2. Unzip file
3. Open Google Chrome or Any Chromium-based browser and go to Manage Extensions
4. Turn ON developer mode
5. Click on Load unpacked button and select the unziped file
6. Now make sure to PIN the chrome extension so that it is visible at all times
7. Go to zepto and search for anything
8. Click on Zepto extension to see the products sorted by discount
9. You can also open any brand page and do the same.

Extension Supports - Zepto Search and Brand Pages. May add more in future.

Known Bug(s) - 

1. Sometimes if you do a second search the product from old search are also shown in the same table. To avoid this do a hard refresh Ctrl + F5.
